# Zuzo Robotics

Marketing site and order-book app for **Zuzo Robotics** — a decentralised physical-AI data network on Bittensor that turns egocentric human video into verified action and trajectory data for training robot policies.

## Pages
- **`index.html`** — marketing site (Network) + order book (Launch app), single-file, client-side routed.
- **`design-system.html`** — the canonical design system. Build every new feature from these tokens and components.

## Design system
White canvas, near-black ink, orange (`#EC450D`) as a sparing point-accent. Inter for type, JetBrains Mono for data. Full tokens live in the `:root` block of each file and are documented in `design-system.html`. Before shipping any new feature, confirm: neutral canvas, institutional full-sentence copy, and components reused from the design system.
